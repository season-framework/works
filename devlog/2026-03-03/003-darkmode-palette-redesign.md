# 다크모드 색상 팔레트 재조정 (가시성 향상)

- **ID**: 003
- **날짜**: 2026-03-03
- **유형**: 버그 수정

## 작업 요약
다크모드의 회색/파란색 계열 색상이 촌스럽고, border가 너무 뚜렷하여 가독성을 저해하는 문제를 전면적으로 재조정. 총 119건의 다크모드 셀렉터로 확장.

## 주요 변경 사항

### 1. 배경색 (Warm Dark)
- 순수 zinc gray → 미세한 따뜻한 톤으로 전환
- `bg-white`: #18181b → #111113 (더 깊은 톤)
- `bg-neutral-100`: #1f1f23 → #1a1a1e (표면 구분 부드럽게)

### 2. 텍스트 (가독성 개선)
- 밝기 계층을 재조정하여 더 자연스러운 위계
- neutral-700: #d4d4d8 → #b4b4be (부드러운 전환)
- neutral-500: #71717a → #6b6b78 (머티드하지만 읽기 가능)

### 3. Border (대폭 완화)
- 100/200: #27272a → #1c1c23 (배경과의 대비 대폭 감소)
- 300: #3f3f46 → #28282f (확연히 부드러움)

### 4. 파란색 → 인디고 시프트
- blue-500: → #5b6cf0 (덜 채도 높은 인디고 톤)
- blue-600 텍스트: → #818cf8 (파스텔 인디고, 눈에 편안)
- bg-blue-50: → rgba(99,102,241,0.06) (미묘한 인디고 틴트)
- focus 포커스 색상: → #5b6cf0 (통일)

### 5. 추가 오버라이드
- Hover: opacity 기반으로 더 부드럽게 (0.08 → 0.04~0.06)
- Shadow: 더 깊고 자연스러운 그림자
- Badge/Tag: 다크에서 배경 과도한 대비 완화
- Checkbox accent: 인디고 통일
- Black overlay: 0.3 → 0.5 (다크 배경에서 더 확실한 딤)
