# 멘션 텍스트 배지 스타일 적용

- **ID**: 002
- **날짜**: 2026-03-06
- **유형**: 버그 수정

## 작업 요약
이슈보드 메시지에서 `@이름` 멘션이 일반 텍스트와 동일하게 표시되던 문제 수정. `display.ts`의 멘션 감지 정규식이 HTML 태그까지 캡처하여 HTML 구조가 깨지는 버그 수정. 정규식을 `(^|[\s>])@([^\s<>&]+)`로 개선하여 HTML 태그를 안전하게 처리하고, 시각적으로 구분되는 파란색 배지(둥근 필 배경) 스타일 적용.

## 변경 파일 목록

### Portal Package (works)
- `src/portal/works/libs/struct/display.ts`: `markdown()` 메서드 멘션 정규식 수정 — `@(\S+)` → `(^|[\s>])@([^\s<>&]+)`, 배지 스타일 개선 (파란 배경, 둥근 모서리, bold)
